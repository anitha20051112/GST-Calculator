# GST Calculator

A simple and user-friendly GST Calculator built using HTML, CSS, and JavaScript.

## Features

* Calculate GST instantly
* Supports GST rates: 5%, 12%, 18%, and 28%
* Displays GST amount and total amount
* Clean and responsive user interface
* 100% free to use

## How It Works

1. Enter the amount.
2. Select the GST rate.
3. Click the **Calculate** button.
4. View the GST amount and total amount instantly.

## Technologies Used

* HTML5
* CSS3
* JavaScript
